## 函数     Worksheet

函数能将结果返回给调用者，子过程不能返回结果给调用者。

```vb
Option Explicit
Sub toUSDs()
    Dim level
    level = customOpe(Cells(8, 6))
    Cells(7, 6) = level    
End Sub

Function customOpe(ars)
Dim score, level
score = ars

If score < 30 Then
    level = "soft"
ElseIf score > 30 Then
    level = "hard"
Else
    level = "mix"
End If

customOpe = level
End Function
```

## Worksheet 对象

```vb
Option Explicit
Sub abc()
    Dim a, s
    
    s="ABCDEF GHIJK MNL OPQ"
    
    a = Trim(s)
    a = Len(s)
    a = Replace(s, " ", "1")
    a = UCase(LCase(s))
    a = Right(s, 2)
    a = Mid(s, 3)
    a = InStr(10,s, "ab")
    
    MsgBox "[ " & a & " ]"

End Sub
```
